# Prediksi Hasil Panen Kelapa Sawit

Aplikasi ini dibuat menggunakan Python dan Streamlit untuk memprediksi hasil panen kelapa sawit berdasarkan data yang tersedia.

## Fitur
- Menampilkan dataset dalam bentuk tabel interaktif
- Opsi melihat preview atau seluruh data
- Prediksi hasil panen (Tinggi / Rendah)
- Menggunakan perbandingan dengan rata-rata data
- Visualisasi data menggunakan grafik

## Teknologi
- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib

## Dataset
Dataset berisi data kelapa sawit yang mencakup variabel lingkungan seperti NDVI, curah hujan, suhu, kelembaban, pupuk, luas lahan, serta hasil panen dalam satuan ton per hektar.
